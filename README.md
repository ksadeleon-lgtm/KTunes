##🎵 Music Player
A simple and elegant music player built using HTML, CSS, and JavaScript. This player have albums and explore to look for songs, providing a seamless and organized listening experience.


##🚀 Features
✅ Play, Pause, Next, and Previous song controls
✅ Dynamic album and playlist loading from info.json
✅ Album cover images for each playlist (image.jpg)
✅ Responsive and user-friendly UI
✅ Displays album art and song name and artist
✅ Smooth audio playback
✅ Displays explore button that contain the genre 
✅Music Albums View — Browse and navigate through five albums
✅Favorites — Click the heart in the player to automatically add a song to favorites
✅Recently Played — Tracks the songs you've played with a reset button
✅Search Bar — Search for a song by title or artist in real time
✅Progress Bar — Click or drag to seek through a song
✅Volume Control — Slider for users to cotrol the volume

##👩‍💻 Technologies Used
HTML5 - Structure of the music player
CSS3 - Styling and layout design
JavaScript - Functionality and dynamic interactions

##🌟 How to Use------------
Open index.html in a browser.
Enjoy your music! 🎶
Each album or playlist folder contains an file NAME, which provides details about the playlist.

"title" → The playlist or album name displayed in the player.
Additionally, every album folder should contain an image.jpg file.


This serves as the album cover for the playlist.
You can upload any image, but it must be named image.jpg for the player to display it correctly.
🔧 Customization
🎼 Add new albums and songs inside the songs/ folder.
🎨 Replace image.jpg with any album cover image (keep the same name).
🎭 Customize the UI in style.css.

👥Target Users
The target users of this web music player are students, working individuals, and casual music listeners who want a simple and easy to use platform for listening to music while studying, working, relaxing, or commuting. The application is designed for users who prefer quick access to playlists, smooth music playback, and a clean interface without unnecessary distractions and too much features.
